        GUIDA ALLE BOX MP4                        

ftyp  → File Type Box                             
        Specifica il tipo di file e i brand       
        compatibili (es. isom, mp42, etc.)        
                                                  
moov  → Movie Box                                 
        Contiene i metadati del file, come        
        durata, tracce video/audio, header        
                                                  
mdat  → Media Data Box                            
        Contiene i dati grezzi audio/video        
        (frame e suoni compressi)                 
                                                  
udta  → User Data Box                             
        Sezione opzionale con metadati utente:    
        titolo, autore, commenti, tag
